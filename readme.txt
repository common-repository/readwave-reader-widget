== Description ==

This plugin allows you to embed the readwave reader widget in pages and posts on your wordpress blog. 

Upload your project on Readwave.com and then copy the project id from the url of your project page. The id is the number after the "_s". In the following example the id would be 1234

* http://www.readwave.com/my-awesome-story_s1234

You can add a reader widget for this story by using the readwave shortcode as follows:

* [readwave]1234[/readwave]

== Installation ==

This section describes how to install the plugin and get it working.

1. Download the plugin from: http://readwave.com/dl/wordpress/readwave.zip
1. Upload the files to your plugin directory `/wp-content/plugins/`
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add the shortcode to the body of your post or page

== Frequently Asked Questions ==

= Where can I upload my stories =

You can upload stories by creating an account on readwave.com

= How do I get my story id =

You can get the story id from the end of your story url.

= How do I embed my story in a post or page =

You can embed your story using the [readwave][/readwave] shortcode around your story id.
